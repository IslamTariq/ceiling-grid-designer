# Ceiling Designer

Ceiling designer/editor built with React and TypeScript.



# Features

Add, remove, and move components on a grid
Pan and zoom functionality
Support for 5 component types (Lights, Air Supply, Air Return, Smoke Detectors, Invalid Grid)
Standard 0.6m x 0.6m grid cell size



## How to run

Clone repo from git
npm install
npm run dev



# How to use features

Place Component: Select component type, click empty cell
Remove Component: Click existing component
Move Component: Drag component to new cell
Pan: Middle/Right mouse button + drag
Zoom: Mouse wheel or zoom buttons
Reset: Click Reset button to clear grid



# Tech stack

React 19 + TypeScript
Vite
Canvas
Material Icons
